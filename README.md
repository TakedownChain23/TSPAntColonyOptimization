To run our project:

1. Install dotnet 10.0.0
2. Clone our repo locally
3. Use a C# IDE (Rider or Visual Studio) to build and run the program, and run the tests

   OR

1. Install dotnet 10.0.0
2. Clone our repo locally
3. Navigate to the project directory in command line
4. Run the 'dotnet restore' command to restore any dependencies
5. Run the 'dotnet build' command to build the project
6. Run the 'dotnet run --project TravellingSalesman' command to run the program
7. Run the 'dotnet test' command to run the tests

Note: we have some test graphs in a file called TSPLIB, it run our program with different test graphs you have to change the target filename in the Program.cs file to whichever test file you want to use.
